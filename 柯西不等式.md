# 柯西不等式

## 证明
### 方法一：数学归纳法
这个方法是容易的，最后一步需要用到均值不等式
主要思想是将两个和(n)的乘积拆成和(n-1)的乘积，之后处理余项
对余项使用均值不等式放缩
### 方法二：参数化方法
见：谢惠民
取等条件的证明：
当原函数取极值 0 时，不等式取等
此时，原来的 n 个平方数加和结果为 0，亦即 n 项全部都取 0，由此得到取等条件

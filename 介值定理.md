# 介值定理

## 概述

### 定理内容

若 $f(x)\in C_{[a,b]}$ 且有 $f(a)\cdot f(b)<0$ ，则 $\exists \xi\in(a,b)$ 满足 $f(\xi)=0$ 

### 分析

1. 该定理依赖于 [[实数的完备性定理]] ，对于 $\mathbb{Q}$ 不成立
2. 该定理依赖于函数的 [[连续性]] 性质
3. 该定理只叙述了 $\xi$ 的 **存在性**，也即 $\xi$ 可能 **不唯一** 

## 证明

### 证明一：使用 [[闭区间套定理]] 证明

这里构造闭区间套使用了 [[波尔查诺 (Bolzano) 二分法]] 

不妨有：$f(a)<0,f(b)>0$ 

先取 $[a_{1},b_{1}]=[a,b]$ ，取 $x_{1}=\frac{a_{1}+b_{1}}{2}$ 

若 $f(x_{1})=0$，证毕
若 $f(x_{1})>0$，则取 $[a_{2},b_{2}]=[a_{1},x_{1}]$
若 $f(x_{1})<0$，则取 $[a_{2},b_{2}]=[x_{1},b_{1}]$ 

依此续行，若一直没有找到 $f(x_n)=0$，则得到一系列的闭区间套，满足：
1. 区间长度趋于零（由 Bolzano 二分法保证）
2. 恒有：$f(a_n)<0<f(b_n)$ 

对于上述闭区间套使用闭区间套定理，则有：
$$
\exists! \xi\in \bigcap_{k=1}^{\infty}[a_n,b_n],\lim_{ n \to \infty } a_n=\lim_{ n \to \infty } b_n=\xi
$$
**以下使用到了连续函数的性质**：极限与函数符号 **可以换序** 
对于 2 两边同时取极限，得到：
$$
\lim_{ n \to \infty } f(a_n)=f(\lim_{ n \to \infty } a_n)=f(\xi)\leq 0\leq \lim_{ n \to \infty } f(b_n)=f(\lim_{ n \to \infty } b_n)=f(\xi)
$$
也即：$f(\xi)=0$，证毕

### 证明二：使用确界定理证明

使用 [[确界定理]] 证明

考察 $[a,b]$，设集合：$\mathbb{F}=\{x|f(x)<0,x\in[a,b]\}$ ，显然非空，并且有上界 b
由确界定理，$\mathbb{F}$ 有上确界，记为 $c=\sup\mathbb{F}$ ，下证 c 符合条件：

考虑从两方面证明：一方面 $f(c)\leq 0$，另一方面 $f(c)\geq 0$ 

#### 从左侧逼近说明

一方面，由上确界的性质，必能够取一个数：
$$
\exists x_n\in \mathbb{F},\ \text{s.t.}\ c-\frac{1}{n}<x_n\leq c
$$
对两侧使用夹逼定理，得到：
$$
\lim_{ n \to \infty } \left( c-\frac{1}{n} \right) = c \leq \lim_{ n \to \infty } x_n \leq c
$$
由连续性条件，可以套上函数符号，则有：
$$
\lim_{ n \to \infty } f(x_n)=f(c)
$$
而对于集合 $\mathbb{F}$，由构造方式，显然有：
$$
f(x_n)<0
$$
取极限得到：
$$
f(c)=\lim_{ n \to \infty } f(x_n) \leq 0
$$
#### 从右侧逼近说明

另取一个数 $x_n=c+\frac{1}{n}$ ，$\exists N\in \mathbb{N},\forall n>N,x_n<b$ 

另一方面，由于 $x_n>c$ 故有 $x_n\not\in \mathbb{F}$ ，则有 $f(x_n)>0$

对两侧取极限，由连续性条件有：
$$
\lim_{ n \to \infty } f(x_n)=f(\lim_{ n \to \infty } x_n)=f(c)\geq 0
$$

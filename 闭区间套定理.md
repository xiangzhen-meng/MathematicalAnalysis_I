# 闭区间套定理
## 叙述
### 等价叙述一：集合语言叙述

若 $\{I_{n}\}$ 为闭区间套，则其交集不为空；
若其长度满足 $\exists \epsilon >0,\forall n\in\mathbb{N},b_{n}-a_{n}<\epsilon$ 则其交集为单点集
### 等价叙述二：数列极限叙述

若闭区间序列满足条件：
$$
[a_{n},b_{n}],an\leq a_{n+1},b_{n}\geq b_{n+1}
$$
则存在 $\xi$ 使得 $a_{n} < \xi < b_{n},n\in\mathbb{N}$ 
进一步地，如果区间长度极限为 0，则上述 $\xi$ 唯一
## 证明
### 证明一：利用实数的十进制小数表示证明

构造性证明：
考虑每一个闭区间的下界 an 序列，将其由小到大排序
逐位地，取 an 序列的最大值，将某一位 $\alpha_{i}$ 的最大值所在的第 n 个数记作 $\alpha_{i},n_{i}$ 
最后取得的某一个数，称作 c

下证**存在性**：c 是闭区间的交
**一方面**，由取法，显然 c 是 an 的上界
**另一方面**，想要证 c 是 bn 的下界，采用反证法：
假设有一个 b 小于 c，则必然能够找到小的那一位，记作第 k 位
**因为 b 小于 c，所以合理推测会存在一个比 b 大的 a，那么接下来操作的目的在于取出一个比 b 还要大的 a**
则这个 b 也可以记成 $\beta_{k} ,n_{1}$ 
必然能够找到一个在 k+1 位不是最大，而在第 k 位是最大的 a 序列中的数，那么这个数也比 b 大，设为第 n2 个数
**注意**：此时我们已经发现了一个比 b 大的 a，那么直觉上看来已经矛盾了
如果 n1 > n2，则：
必定能够找到第 n2 个 b ，则他在第 n2 个 a 的右边，则此时 \[an1, bn1] 与 \[an2, bn2] 不交，与闭区间套矛盾
如果 n1 <= n2，则：
bn2 <= bn1，又因为 bn1 < an2，所以 bn2 < an2，这与区间 \[an, bn] 矛盾
综上，存在性证毕

下证**唯一性**：
考虑反证法：若不然，假设有 c1, c2 均满足条件，不妨：
$$
\forall n\in\mathbb{N},a_{n} \leq c_1 < c_2 \leq b_n
$$
则
$$
b_n - a_n \leq c_2 - c_1 < \epsilon
$$
因此
$$
c_2 \geq b_n + (c_1 - a_n) \geq b_n
$$
矛盾！故唯一性成立
**注意**，此处是一个经典的 epsilon 相关操作，在 [[确界定理]] 的证明中也有使用
### 证明二：利用序列极限证明

我们将两个极限如下表示：
$$
\lim_{ n \to \infty } a_n = \lim_{ n \to \infty }b_n := c 
$$
显然有：c 是序列 an 的上确界，是序列 bn 的下确界
下面证明：c 的唯一性
假设存在 c‘ 使得：$a_n \leq c \leq b_n$，对其使用夹逼准则得到：
$$
\lim_{ n \to \infty }a_n = \lim_{ n \to \infty }b_n = c' =c
$$
故唯一性成立

## 推广和例子

### 尝试推广：开区间套行不行？

**不行**：虽然能够保证两个区间端点的极限是那个数，但是最后那个集合应该是空集，该元素不含在集合里面
例子：
$$
a_n = 0,\ b_n = \frac{1}{n},\ c \in (a_n, b_n)
$$
可以证明得到：
$$
\lim_{ n \to \infty }b_n = \lim_{ n \to \infty } a_n = c = 0,\ I_n = \emptyset 
$$

证明：考虑使用反证法
若不然，则：
$$
\exists k \in \bigcap_{n = 1}^{\infty} (a_n, b_n) \iff \forall n\in\mathbb{N},\exists\ k \ s.t. k < \frac{1}{n}
$$
由实数的阿基米德性质推论 1 ，只要取：
$$
N > \frac{1}{k},N\in \mathbb{N}
$$
即得矛盾，证毕

### 例子 2.1 对于非单调序列的控制手法

闭区间套定理给出了一种用 **单调序列** 逼近 **非单调序列** 的手法

这种手法是通过控制 **上界序列** 和 **下界序列** 两个序列，夹逼得到原序列

>[!question] 例子 2.1
>数列 $x_n$ 满足：$$
>x_{1}=a,x_{2}=b, x_n=\frac{1}{2}\cdot (x_{n-1}+x_{n-2})
>$$求证：$$
>\exists \lim_{ n \to \infty } x_n
>$$并求出极限的值

#### 方法一：通过闭区间套定理逼近

若 a=b，则是平凡的

若 $a\neq b$ ，尝试构造 **闭区间套**：

构造**下界数列**：$a_n=min(x_n, x_{n+1})$
构造**上界数列**：$b_n=max(x_n, x_{n+1})$

显然有：$a_n\leq x_n\leq b_n\ (不能同时取等),\ a_{n}\leq x_{n+1}\leq b_{n}\ (不能同时取等)$

想要利用闭区间套定理求出极限，有两件事情要干：
1. 证明上下界数列的**单调性**
2. 证明**区间长度趋于零**

先证单调性：

考察：
$$
\begin{align}
a_{n+1}-a_n&=min(x_{n+1},x_{n+2})-min(x_{n},x_{n+1}) \\
&=min( x_{n+1}, \frac{1}{2} (x_{n+1}+x_n))-min(x_{n},x_{n+1}) \geq 0
\end{align}
$$
可以这样理解：最小值的最大值一定出现在均值处
严谨的证明只要讨论 $x_n 与x_{n+1}$ 的大小关系即可

故下界数列递增
同样地，可以证明上界序列递减

此时，我们已经构造出了**闭区间套**，也即第一件事干完了
下面只要证明区间长度趋于 0即可

$$
\begin{align}
\lim_{ n \to \infty } |a_n-b_n| &= \lim_{ n \to \infty } |x_{n+1}-x_n| \\
\end{align}
$$
而经过迭代，有：
$$
\begin{align}
|x_{n+1}-x_n|=&| \frac{1}{2}(x_n + x_{n-1})-x_n| \\
=& \frac{1}{2}|x_n-x_{n-1}| \\
=&\dots \\
=& \frac{1}{2^{n-1}}|x_{2}-x_{1}|
\end{align}
$$
显然有长度趋于 0 

故由闭区间套定理知，序列极限存在，令为 c

下面求 c 的值

考虑使用两边同时取极限的方法求，但是原来的递推关系两边取极限之后直接得 0 了
这样就说明不能直接对递推式取极限，需要得到**常数**：
将原来的数列累加，得到：
$$
x_{n+1}-x_{2}=-\frac{1}{2}(x_n-x_{1})
$$
这回两边同取极限即可，得到答案：
$$
\lim_{ n \to \infty } x_n=\frac{a+2b}{3}
$$

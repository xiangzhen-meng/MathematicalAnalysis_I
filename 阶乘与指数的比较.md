# 阶乘与指数的比较

>[!question] 问题：
>求证$$
>y_n = \frac{c^n}{n!} 是无穷小序列，其中\ c>0
>$$

这里使用了 [[an epsilon of room]] 的思想
判断主项：当 n 趋近于无穷的时候，n 的后无穷多项每一项都远大于 c，而仅有前 c 项小于等于 c，因此：对于前有限多项（多于前 c 项）和后无穷多项分段估计

取定 $m > c,\ n>m \implies n>c$，故有：
$$
\begin{align}
y_n&=\frac{c^{m+1}}{m!}\cdot \frac{c^{n-m-1}}{n(n-1)(n-2)\dots(n-m+1)} \\
&=\frac{c^{m+1}}{m!}\cdot \frac{1}{n}\cdot \frac{c}{n-1}\cdot \frac{c}{n-2}\cdots \frac{c}{n-m+1} \\
&<\frac{c^{m+1}}{m!}\cdot \frac{1}{n}
\end{align}
$$
而 $\frac{1}{n}$ 是无穷小数列，又有 $y_n>0$ ，因此 $y_n$ 是无穷小数列

# 皮亚诺公理 (Piano Axioms)

### 内容
1. PA1: 0 是自然数
2. PA2: 如果一个数是自然数，他的后继也是自然数
3. PA3: 0 不是任何自然数的后继
4. PA4: 不同自然数的后继不同 <=> 两个自然数的后继相同，则这两个自然数相同
5. PA5: （归纳公理）对于某个性质 $P(n)$ ，如果 $P(0)$ 为真，且如果 $P(n)$ 为真推出 $P(n的后继)$ 为真，则 $\forall x\in \mathbb{N},P(x)$ 为真
### 理解
#### PA3 和 PA4 的作用
PA3、PA4 都是为了防止后继的 **绕回** 而设置的：
PA3: 防止某个数的后继是 0，但是光是这样还是不够的
PA4: 防止某个数的后继是他前面的数或是他自己
#### PA5 的作用
如果没有归纳公理，我们可以对自然数集引入一组新的病态数：
$$
\{a, b\} \ satisfy: S(a) = b,\ S(b) = a
$$
这样的 a 和 b 也是自然数，并不违背前四条公理

为了防止这样的事情发生，我们选择引入归纳公理：
考察性质 P，我们将永远无法通过 P(0) 为真推导出 P(a) 为真，而由归纳公理，这保证了 a 不是一个自然数，亦即自然数永远只有 0, 1, 2, 3, ...
#### 某些其他的定义
有些人（工院老师们和苏联数学家们）认为 0 不是自然数。这也可以。只要稍稍修改一下公理的部分细节即可，会得到一个拥有同样性质的体系

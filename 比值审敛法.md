# 比值审敛法

## 概述

考虑一个级数：
$$
\sum_{n=1}^{\infty} a_n.
$$
比值审敛法的核心是考察数列相邻项的比值的极限。定义数列 $a_n$ 的比值：
$$
L = \lim_{n \to \infty} \left| \frac{a_{n+1}}{a_n} \right|.
$$

根据比值 $L$ 的值，可以得出以下结论：

若 $L < 1$：级数 **绝对收敛**。即，级数 $\sum_{n=1}^{\infty} a_n$ 收敛
若 $L > 1$：级数 **发散**。即，级数 $\sum_{n=1}^{\infty} a_n$ 发散
若 $L = 1$：此时，**无法得出结论**，需要使用其他方法进一步分析

## 例题
### 问题 1 比值审敛法与根值审敛法

>[!question] 问题 1 
>对于正项数列 $\{a_n\}$ $$
>若\lim_{ n \to \infty } \frac{a_{n+1}}{a_n}=a, 求证\ \lim_{ n \to \infty }  \sqrt[n]{ a_n } = a
>$$

这个问题是非常重要的，因为其给出了一个事实：**用比值审敛法能做的事情，用根值审敛法一定能做，反之则不一定** ，也即 根值审敛法 是一个比 比值审敛法 更强的定理

#### 证法一：用 $\varepsilon-N$ 直接干

利用 [[an epsilon of room]] 的思想，对于无限项的乘积即使是可以分子分母消去，也是不好处理的，故而要取定一个 N

取定 N，使得 $\forall \varepsilon>0,\forall n >N, |\frac{a_{n+1}}{a_n} - a|<\varepsilon \implies a-\varepsilon< \frac{a_{n+1}}{a_n}<a+\varepsilon$

$$
\begin{align}
\sqrt[n]{ a_n } &= \sqrt[n]{ \frac{a_n}{a_{n-1}}\cdot \frac{a_{n-1}}{a_{n-2}}\dots \frac{a_{N+1}}{a_N} \cdot a_N } \\
&\leq \sqrt[n]{(a+\varepsilon)^{n-N}\cdot a_N} \\
&=\sqrt[n]{ a_N }\ \cdot (a+\varepsilon)^{1-N/n}
\end{align}
$$

显然有：左边那项的极限为 1 ，只要证右边的极限为 a 即可

也即证：
$$
\forall \varepsilon'>0,\exists N_1,\forall n>N_1,\left|(a+\varepsilon)^{1-N/n}-a\right|<\varepsilon'
$$
在这里打个马虎眼，直接认为 $(a+\varepsilon)^{1+N/n}\to a+\varepsilon$ 

**显然直接干不太成功**

#### 证法二：使用夹逼准则

这里使用了 [[柯西命题-序列均值极限#推论：无穷小数列的几何、调和、平方均值仍是无穷小]] 

一方面，考虑放大的方向，也即 [[均值不等式#AM-GM 证明]]
补充定义：$a_{0}=1$ ，则有：
$$
\begin{align}
\sqrt[n]{ a_n }&=\sqrt[n]{ \frac{a_n}{a_{n-1}}\cdot \frac{a_{n-1}}{a_{n-2}}\dots \frac{a_2}{a_1}\cdot a_1 } \\
&\leq \frac{\frac{a_n}{a_{n-1}}+ \frac{a_{n-1}}{a_{n-2}}+\dots+ \frac{a_2}{a_1}+ \frac{a_1}{a_{0}} }{n}
\end{align}
$$
而分子上面每一项为极限趋于 a 的序列，由柯西命题，RHS 极限为 a

另一方面，考虑缩小的方向，显然由推论，RHS 对应的调和均值极限也为 a，故有
$$
\begin{align}
\sqrt[n]{ a_n }&\geq \frac{n}{\frac{a_{n-1}}{a_n}+ \frac{a_{n-2}}{a_{n-1}}+\dots+ \frac{a_1}{a_2}+ \frac{a_{0}}{a_1} }
\end{align}
$$
由夹逼准则，证毕

### 问题 2：根值审敛法不成功时，使用比值审敛法

>[!note] 问题 2
>求证：$$\lim_{ n \to \infty } \sqrt[n]{ \frac{n^n}{n!} }=e$$

这就是属于用根值不好干的。下面用比值审敛法干他：

令 $a_n = (n^n)/(n!)$ ，则有：
$$
\frac{a_n}{a_{n-1}}=\frac{n^n}{n!}\cdot \frac{(n-1)!}{(n-1)^{n-1}}=\frac{n^{n-1}}{(n-1)^{n-1}}=\left( \frac{n}{n-1} \right) ^{n-1}=\left( 1+ \frac{1}{n-1} \right)^{n-1} 
$$
这是第二个重要极限，因此有：
$$
\lim_{ n \to \infty } \frac{a_n}{a_{n-1}}=e
$$
由上述夹逼定理，可以知道 
$$
\lim_{ n \to \infty } \sqrt[n]{ \frac{n^n}{n!} }=\lim_{ n \to \infty } \frac{a_n}{a_{n-1}}=e
$$
这也说明了：$n^n$ [[增长速度的比较|增长速度]] 快于 $n!$ 

这个命题其实跟 [[对阶乘的估计#用斯特林公式估计]] 是同一回事
